# 真实数据实验替换方案

目标：把当前 33 个核心案例从“演示型小数组”替换为“真实开放数据 + 小样本在线实验 + 完整离线复现脚本”。

状态：第一批 12 个“核心案例版”已落地到网页实验覆盖层，覆盖 MTA GTFS、NYC TLC、NYC 事故、UCI Bike、Metro I-94、PeMS-SF、GeoLife 和 KITTI 等来源。

## 执行原则

1. 在线实验只放小样本，建议每个案例 5KB-200KB，保证浏览器内 Python 能快速运行。
2. 每个样本保留来源、下载日期、字段说明和许可说明。
3. 大数据下载、模型训练、图像/视频处理放到离线进阶项目。
4. 不适合代码化的小节只做数据认知，不强行制造重复实验。
