# 交通数据挖掘开放课程

《交通数据挖掘理论与应用》的静态电子课程网站，覆盖第 1 至第 8 章的开放数据实践。

发布地址：`https://lilinchao.github.io/traffic-book-practice/open-course/`

## 本地预览

```powershell
python -m http.server 8000
```

打开 `http://localhost:8000`。

## 内容

- 33 个浏览器内可编辑、可运行的 Python 核心案例，覆盖第 1 至第 8 章中适合动手的关键小节
- 章节筛选、小节编号、难度与学习时长、实践目标、修改挑战和知识点提示
- 柱状图、折线图、回归图、聚类图和热力图五类即时结果
- 当前实验代码一键下载
- 六个连接开放数据的完整离线进阶项目
- 课程首页、学习路径、数据来源和使用边界
- 桌面端和移动端响应式布局

浏览器实验通过 Pyodide 在本地运行，不上传学生代码，也不需要后端服务器。
