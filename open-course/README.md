# 交通数据挖掘开放课程

《交通数据挖掘理论与应用》的静态电子课程网站，包含第3至第8章的开放数据实践。

发布地址：`https://lilinchao.github.io/traffic-book-practice/open-course/`

## 本地预览

```powershell
python -m http.server 8000
```

打开 `http://localhost:8000`。

## GitHub Pages

本页面作为现有 `traffic-book-practice` 站点的 `open-course/` 子页面发布。

## 内容

- 课程首页与学习路径
- 六个实践模块
- 可下载 Python 脚本
- 开放数据来源和使用边界
- 桌面端和移动端响应式布局
