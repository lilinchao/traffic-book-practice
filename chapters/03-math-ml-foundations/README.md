# 03 数理理论基础

本章实践目标：用计算实验连接数学概念与交通场景，建立微积分、概率、线性代数、优化和机器学习基础的直觉。

## Goals

- 用数值微分计算交通速度时间序列的变化率
- 用泊松分布和负二项分布拟合交通事故计数数据
- 用矩阵运算处理时间 x 检测器的速度矩阵
- 用梯度下降求解线性回归模型
- 用时间序列友好的方式划分训练集与测试集
- 计算 MAE、RMSE、MAPE 并解释交通含义
