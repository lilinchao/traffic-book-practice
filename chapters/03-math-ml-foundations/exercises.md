# Exercises

## Basic

1. 给定一段交通速度时间序列，用数值微分（前向差分、中心差分）计算每个时刻的速度变化率，找出变化率最大的时刻并解释其交通含义。
2. 生成一组符合泊松分布的事故计数数据（lambda=5），绘制直方图并叠加理论概率质量函数。
3. 构造一个 24 x 10 的速度矩阵（24 小时 x 10 个检测器），计算每小时的平均速度向量，并求该矩阵的秩。
4. 用 numpy 实现前向传播和损失计算：给定特征矩阵 X 和标签 y，计算线性模型 y_hat = Xw + b 的 MSE 损失。

## Advanced

1. 对同一组真实事故计数数据分别拟合泊松分布和负二项分布，用卡方检验比较两个拟合的优劣。
2. 对速度矩阵做 SVD 分解，用前 k 个奇异值重建矩阵，计算重建误差随 k 的变化，解释 k 的交通含义。
3. 实现批量梯度下降求解线性回归，在学习率 0.001、0.01、0.1 下分别运行 200 轮，记录每轮损失，绘制收敛曲线。
4. 按时间顺序划分训练集和测试集（前 80% 训练，后 20% 测试），用线性回归预测交通流量，计算 MAE 和 RMSE。

## Challenge

1. 用拉格朗日乘数法求解约束优化问题：在总预算固定的条件下，选择多个路段的速度限制使总事故期望数最小（写出拉格朗日函数并用数值方法求解）。
2. 实现带 L2 正则化的梯度下降，在训练集和测试集上分别记录损失，绘制学习曲线，找出过拟合发生的正则化系数阈值。
3. 对交通流量预测模型分别计算 MAE、RMSE 和 MAPE，分析三种指标在不同流量水平下的行为差异（低流量时 MAPE 是否会异常偏大？为什么？），并给出选择建议。
4. 设计一个完整的实验流程：生成模拟数据 -> 特征工程（添加滞后特征） -> 时间序列划分 -> 训练线性回归 -> 计算评价指标 -> 分析过拟合，输出一份 300-500 字的实验报告。
