# 第 4 章实践讲义：交通表格型数据分析

本讲义面向学生上机实践使用，不作为书籍正文。案例使用英国 STATS19 交通事故表格数据，目标是学习如何通过模型分析自变量对因变量的影响。

配套 Notebook：

`notebooks/chapter-04/traffic_accident_panel_analysis.ipynb`

## 1. 实践目标

完成本实践后，你应该能够：

- 识别交通表格数据中的因变量、自变量和控制变量。
- 解释事故严重程度、伤亡人数和弱势交通参与者伤亡数三个建模目标。
- 使用逻辑回归分析自变量对严重事故概率的影响。
- 使用泊松回归分析自变量对伤亡人数的影响。
- 使用负二项回归理解计数模型的替代方案。
- 使用零膨胀模型分析零值较多的计数结果。
- 区分“相关性回归解释”和“严格因果识别”。

## 2. 数据集

本实践使用英国 Department for Transport 的 STATS19 Road Safety Data。

数据源：

https://data.dft.gov.uk/road-accidents-safety-data/

本仓库使用：

| 文件 | 说明 |
| --- | --- |
| `data/raw/stats19_collision_2023.csv` | 2023 年碰撞事故表 |
| `data/raw/stats19_casualty_2023.csv` | 2023 年伤亡人员表 |
| `data/processed/stats19_collision_casualty_tabular_2023_sample.csv` | 合并后的教学样本 |

处理后的教学样本每行是一条碰撞记录。

## 3. 变量设计

### 因变量

| 因变量 | 类型 | 模型 |
| --- | --- | --- |
| `severe_or_fatal` | 二分类 | 逻辑回归 |
| `number_of_casualties` | 计数 | 泊松回归、负二项回归 |
| `vulnerable_casualties` | 零值较多的计数 | 零膨胀泊松模型 |

### 自变量

| 自变量 | 含义 |
| --- | --- |
| `speed_limit` | 道路速度限制 |
| `rural` | 是否为乡村地区 |
| `darkness` | 是否为夜间或照明不足环境 |
| `adverse_weather` | 是否为不良天气 |
| `wet_or_icy` | 路面是否潮湿、积雪或结冰 |
| `junction_present` | 是否在路口附近 |
| `weekend` | 是否为周末 |
| `number_of_vehicles` | 涉及车辆数 |

## 4. 建模问题

本案例围绕三个问题：

1. 哪些因素与严重或死亡事故概率更高有关？
2. 哪些因素与一次碰撞造成更多伤亡人数有关？
3. 当弱势交通参与者伤亡数有大量零值时，应该如何建模？

注意：这些问题是因果导向的建模练习，但当前数据和模型不能直接证明因果关系。

## 5. 模型解释

### 5.1 逻辑回归

因变量：

```text
severe_or_fatal
```

模型解释重点：

- `odds_ratio > 1`：对应变量与严重/死亡事故优势比升高相关。
- `odds_ratio < 1`：对应变量与严重/死亡事故优势比降低相关。
- 不能直接解释为政策效果。

### 5.2 泊松回归

因变量：

```text
number_of_casualties
```

解释重点：

- `incidence_rate_ratio > 1`：期望伤亡人数增加。
- `incidence_rate_ratio < 1`：期望伤亡人数减少。
- 需要检查计数数据的离散程度。

### 5.3 负二项回归

负二项模型用于处理计数数据的过度离散。如果泊松模型的方差假设不合适，可以用负二项模型作为替代方案。

### 5.4 零膨胀模型

因变量：

```text
vulnerable_casualties
```

很多碰撞没有行人或骑行者伤亡，因此该变量有大量 0。零膨胀模型把问题拆成两部分：

- 为什么某些观测更可能是结构性零值？
- 非结构性零值情况下，伤亡人数如何变化？

## 6. 因果分析边界

当前案例可以训练“因果分析的思维”，但还不是严格因果识别。

更严格的因果分析需要：

- 明确处理变量，例如速度限制调整、道路照明改善、路口改造。
- 控制混杂因素，例如交通流量、道路等级、天气暴露、执法强度。
- 使用更强识别策略，例如匹配、倾向得分、双重差分、断点回归或固定效应模型。

## 7. 学生任务

### 基础任务

1. 读取处理后的 STATS19 表格数据。
2. 解释三个因变量的含义。
3. 比较不同速度限制下的严重事故比例。
4. 绘制弱势交通参与者伤亡数的零值分布。

### 进阶任务

1. 拟合逻辑回归并解释一个 odds ratio。
2. 拟合泊松回归并解释一个 incidence rate ratio。
3. 比较泊松回归和负二项回归的 AIC。
4. 判断泊松模型是否存在明显过度离散。

### 挑战任务

1. 拟合零膨胀泊松模型。
2. 解释零膨胀部分和计数部分分别回答什么问题。
3. 选择一个自变量，写出可能的混杂因素。
4. 设计一个更接近因果识别的研究方案。
