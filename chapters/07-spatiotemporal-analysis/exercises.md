# Exercises

## Basic

1. 运行 `scripts/generate_synthetic_network.py` 生成合成路网数据，读取速度长表 `data/processed/synthetic_network_speed.csv`，输出数据形状和前 10 行。

2. 将速度长表转换为节点-时间矩阵（行为节点、列为时间步），计算并打印矩阵的形状和缺失比例。

3. 读取邻接矩阵 `data/processed/synthetic_network_adjacency.npz` 和节点坐标 `data/processed/synthetic_network_nodes.csv`，绘制路网拓扑图（节点用坐标定位，边用邻接矩阵连接）。

4. 选取一个节点，绘制其一周内 5 分钟间隔的速度时间序列，标注早晚高峰时段。

## Advanced

1. 计算所有节点的时滞互相关矩阵，找出拥堵传播最强的节点对及其时间延迟。

2. 使用 DTW 距离和 K-means 对路段进行聚类（K=3），分析每类路段的速度模式特征（如高峰形态、波动程度）。

3. 构建基于距离的高斯核邻接矩阵，与拓扑邻接矩阵对比，讨论两种邻接矩阵的适用场景。

4. 拟合 VAR 模型进行多节点速度预测（预测步长为 12 步即 1 小时），计算 MAE 和 RMSE。

## Challenge

1. 实现 XGBoost 时空预测模型，特征包括：历史速度滞后、邻居速度滞后、时间特征（小时、星期）、空间特征（度中心性、邻居均值），与 VAR 模型和历史平均基线比较预测误差。

2. 按空间区域（中心/边缘）和预测步长（1/3/6/12 步）分别计算误差，分析模型在不同条件下的表现差异。

3. 使用 STGCN 或 DCRNN 图神经网络模型进行路网速度预测，与上述所有模型比较，撰写 300-500 字分析结论，必须包含：模型结果对比、空间误差分布特征、模型选择建议。
