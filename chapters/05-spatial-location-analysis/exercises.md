# Exercises

## 基础（4 题）

1. 读取 `data/raw/nyc_motor_vehicle_collisions_crashes_2023_sample.csv`，筛选含有有效经纬度的记录，使用 `geopandas.GeoDataFrame` 构建点几何，并设定 CRS 为 EPSG:4326。

2. 将上述 GeoDataFrame 从 WGS84（EPSG:4326）投影到适合纽约市的 UTM 投影坐标系（EPSG:32618），比较投影前后经纬度坐标与平面坐标的差异。

3. 按行政区（borough）汇总碰撞次数和伤亡人数，将汇总结果与行政区面状几何合并，绘制各区碰撞数量的分级统计图（choropleth map）。

4. 解释空间依赖性（spatial dependence）与空间异质性（spatial heterogeneity）的含义，并举例说明它们在交通事故空间分析中的表现。

## 进阶（4 题）

1. 使用 `libpysal` 分别构建 Queen 邻接权重矩阵和 K 近邻权重矩阵（K=5），比较两种权重矩阵的非零元素比例和连接度分布。

2. 计算各区碰撞率的全局 Moran's I 统计量及其 p 值，判断碰撞率是否存在显著空间自相关。绘制 Moran 散点图并解释四个象限的含义。

3. 计算局部 Moran's I（LISA），绘制 LISA 聚类图，识别高-高聚集区（热点）和低-低聚集区（冷点），并解释其空间含义。

4. 使用 DBSCAN 对碰撞点位进行聚类（eps=500 米，min_samples=15），识别事故热点区域，并在地图上标注聚类结果与噪声点。

## 挑战（3 题）

1. 对碰撞点位进行核密度估计（KDE），选取合适带宽，生成密度栅格，识别事故黑点（密度最高的前 5% 区域），并与 DBSCAN 结果进行比较。

2. 以区级碰撞率为因变量，建立 OLS 回归基线模型；计算 OLS 残差的全局 Moran's I；若残差存在显著空间自相关，分别拟合空间滞后模型（SLM）和空间误差模型（SEM），比较三个模型的拟合优度和残差空间自相关水平。

3. 撰写 300-500 字的分析报告，需包含：空间自相关检验结论、热点/黑点识别结果、空间回归模型选择依据，以及本案例中可改进的空间分析方向（如 MAUP 问题、空间尺度选择、GWR 扩展等）。
